Ir a la carpeta del repositorio

Actuarlizar el repositorio DINNOSOFOT: git pull

Agregar archivo nuevo: git add .     (NOTE EL PUNTO) OJO: Aunque cree un nuevo archivo directamente en la carpeta se debe hacer git add . 

Ver status de archivos agregados: git status

Agregar el commit:  git commit -m "Aquí escribes el commit"  OJO: Es obligatorio hacer el commit

Finalmente para subirlo: git push

Para borrar algún archivo: git rm miarchivo.extencion

Para borrar una carpeta: git rm -r micarpeta

OJO: Después de borrar se debe hacer un commit y push
